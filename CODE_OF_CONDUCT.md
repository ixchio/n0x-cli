# Contributor Covenant Code of Conduct

## Our Standards

This is a technical project. We expect contributors to:

- Be direct and constructive in feedback
- Focus on the code, not the person
- Accept that maintainers make final calls on direction
- Keep discussions on-topic

Behaviour that is not acceptable:

- Harassment, personal attacks, or sustained disruption
- Spam or off-topic promotion

## Enforcement

Issues will be closed and users blocked at maintainer discretion. No appeals process.

## Scope

This code of conduct applies to the GitHub repository, issues, and pull requests.
